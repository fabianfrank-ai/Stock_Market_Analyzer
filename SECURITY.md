## Reporting a Vulnerability

If you discover a security issue in this project, please report it responsibly. You can do this by:

- Sending a message to one of my contacts on my profile  
- Clearly describing the issue, steps to reproduce, and any potential impact.

## Supported Versions

- Only the latest version of the project is actively maintained. Please make sure to use the newest release.

## Security Updates

- Security fixes will be released as quickly as possible. Keep your installation up to date.
